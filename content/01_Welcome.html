Styles for Basic Elements

This theme comes with predefined base styles for nearly all HTML elements, based on the Bourbon SASS library.

Webfonts

Webfonts via Googlefonts. Per Default, Montserrat and Lato are anabled. This can be changed in sass/variables.scss

Check it out

Responsive Grid

Fully responsive Grid based on Bourbon NEAT. Default breakpoints are:

  • 1088px (using 12 column grid)
  • 768px (using 9 column grid)
  • 480px (using 6 column grid)
These can be changed in sass/config.scss

Check it out

Below the Mobile breakpoint the standard Menu is substituted by a mobile drawer navigation using snap.js.
